# Daughter Math Game

一个给 4-6 岁儿童使用的本地网页数学启蒙小游戏，重点服务即将上一年级的姐姐。

## 功能

- 姐姐模式：20 以内加减，自己输入答案。
- 姐姐模式会混合出现 `16 + ? = 19`、`16 - ? = 8` 这类求缺数题。
- 姐姐模式会少量出现 `16 ? 9 = 7` 这类找加号/减号的题。
- 姐姐模式会少量出现 `15 - 8 ? 6` 这类比较大小题，练习 `>`、`<`、`=`。
- 答对后播放鼓励语音并自动进入下一题。
- 本次答对 3、5、10 道题时，会播放阶段鼓励语音。
- 答错后清空输入框，播放慢速讲解旁白，并用动画解释计算方法，但不直接报答案。
- 自动记录本次、当天答题数量、答对数量、再想数量，并把错题放入本地错题本。
- 页面视觉采用更卡通的儿童学习卡片风格。
- 支持本地真人录音提示音；没有录音文件时会自动使用系统语音兜底。
- 妹妹模式：保留 10 以内数数和图形分类。
- 所有进度只保存在本机浏览器 `localStorage` 中。

## 本地运行

直接打开 `index.html`，或在当前目录启动静态服务：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/index.html
```

## 文件

- `index.html`：页面结构。
- `styles.css`：界面样式和动画样式。
- `app.js`：题目生成、答题逻辑、语音反馈、动画讲解。
- `assets/audio/`：可选真人录音提示音目录。

## 真人录音

录音文件可以晚点再放。网页会优先播放 `assets/audio/` 里的录音；如果没有对应文件，会退回到系统语音。

建议每个文件只录一句话，前后不要留太长空白。推荐格式是 MP3、M4A 或 WAV。常用文件名：

- 答对：`correct-1.mp3`、`correct-2.mp3`
- 答错：`try-again-1.mp3`、`try-again-2.mp3`
- 奖励：`reward-3.mp3`、`reward-5.mp3`、`reward-10.mp3`
- 讲解结束：`teach-end-1.mp3`

如果以后直接给原始录音，我可以根据内容识别、分类、重命名并接入网页。
