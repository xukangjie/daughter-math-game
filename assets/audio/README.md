# Audio Pack

这里放给小游戏使用的固定提示音。录音文件还没准备好时，网页会自动使用系统语音兜底。

## 推荐录音方式

- 一个文件只录一句话。
- 每句 1 到 3 秒，语速慢一点，语气轻松。
- 前后不要留太长空白。
- 推荐格式：MP3、M4A 或 WAV。

## 标准文件名

答对提示：

- `correct-1.mp3`
- `correct-2.mp3`
- `correct-3.mp3`
- `correct-4.mp3`
- `correct-5.mp3`

答错鼓励：

- `try-again-1.mp3`
- `try-again-2.mp3`
- `try-again-3.mp3`
- `try-again-4.mp3`
- `try-again-5.mp3`

阶段奖励：

- `reward-3.mp3`
- `reward-5.mp3`
- `reward-10.mp3`

讲解结束：

- `teach-end-1.mp3`
- `teach-end-2.mp3`
- `teach-end-3.mp3`
- `teach-end-4.mp3`
- `teach-end-5.mp3`

如果你直接给原始录音文件，我可以根据内容分类并重命名。
